# Brunch – Restaurant Management System

## Overview
Brunch is a web-based restaurant management system for brunch-focused restaurants. Customers can browse menus, make and cancel table reservations, and read blog posts, while administrators can manage menus, tables, reservations, and user activities efficiently. Built with PHP, MySQL, HTML, CSS, and JavaScript, it provides a secure, responsive, and user-friendly experience.

## Features

### User Features
- Browse daily menus with image or PDF support  
- Reserve tables with ambiance selection (indoor, outdoor, rooftop)  
- Cancel reservations via a dropdown interface  
- Read blog posts about recipes and restaurant updates  
- User authentication (login/signup) with session management  

### Admin Features
- Dashboard with statistics on menus, reservations, and users  
- Manage daily menus: upload, view, and delete  
- Monitor table statuses (booked/available)  
- Track reservations and update their status  
- View registered users and manage roles  
- Update restaurant details and system settings  

### Additional Features
- Responsive design for desktop and mobile devices  
- Real-time updates for menus and reservations  
- Secure authentication with password hashing and session-based access control  

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript, Font Awesome  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP/WAMP (Apache, MySQL)  
- **External Libraries:** Google Fonts (Lora, Poppins), Font Awesome  

## Prerequisites
- XAMPP/WAMP or any server environment with PHP and MySQL support  
- PHP 7.4+  
- MySQL 5.7+  
- Modern web browser
